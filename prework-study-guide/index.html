HTML

  • The head element contains information about the webpage.
  • The header element appears at the top of a webpage and typically includes a title, logo, navbar, or menu.
  • The body element represents the visible content shown to the user.
  • The meta element contains metadata used by the browser and is invisible to the user.
  • The title element contains the title of the webpage.
  • An img element displays an image from a source (src).
  • The section elements are semantic elements that define a section in a document that contains thematic content.
  • The anchor (a) element creates a hyperlink to a reference URL (href).
  • The footer element appears at the bottom of a webpage and generally includes copyright information, slogans, terms of service, references, other menu items, etc.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a symbol initialized with a value that can be used in functions or other places. Variables can also be modified depending on how they are initialized.
  • A function is an step-by-step process that takes inputs, usually a variable, array, or object, through one or more operations which results in an output.
  • Object Oriented Programming or OOP is a system which uses an "object", or a set of key/value pairs, to be used elsewhere.
  • An array is a set of values contained in brackets.
  • A constructor is a special type of function that creates an object, or series of similar objects.
  • A string is a set of one or more characters, usually making up words, that have no numerical value.
  • JSON stands for JavaScript Object Notation.
